- Hum React ka folder banayenge using Vite, Vite ek bundler hai jo k folder banane me help karta hai(Humein jo bhi zaroori files chahiye hongi wo sab vite banake humein de dega.).

-Vite se folder banne ke baad hamare paas multiple folders and files hoti hain, chalo pehle folder ke bare me samajhte hain:
    Node_modules(folder):
    ye folder me wo saari cheezein hoti hain jo humein server se connect karti hain(IS FOLDER ME BILKUL KOI CHULL NHI KARNI HAI.).

    Public(folder):
    is folder me hum apni zaroori cheezein rakhte hain like hamari font files, logos, etc.

    src(folder):
    iske andar hamari main files hoti hain jispe humein kaam karna hai like App.jsx, main.jsx, etc.

    assets(folder inside src folder):
    ye folder bhi public folder ki tarah hota hai isme hum apni logo filles ya kuch or rakh sakte hain.(depands on you aap kasie use karte ho.).

- Ab hum dekhte hain files kon kon si hain or kaise use karni hai:
    hamari app banti hai app.jsx file me jo connect hoti hai main.jsx file se, and main.jsx file index.html file ko control karti hai. Index.html(grandparent)==>>main.jsx(parent)==>>app.jsx(child).

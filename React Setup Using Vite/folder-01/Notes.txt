- Hum React ka folder banayenge using Vite, Vite ek bundler hai jo k folder banane me help karta hai(Humein jo bhi zaroori files chahiye hongi wo sab vite banake humein de dega.).

-Vite se folder banne ke baad hamare paas multiple folders and files hoti hain, chalo pehle folder ke bare me samajhte hain:
    Node_modules(folder):
    ye folder me wo saari cheezein hoti hain jo humein server se connect karti hain(IS FOLDER ME BILKUL KOI CHULL NHI KARNI HAI.).

    Public(folder):
    is folder me hum apni zaroori cheezein rakhte hain like hamari font files, logos, etc.

    src(folder):
    iske andar hamari main files hoti hain jispe humein kaam karna hai like App.jsx, main.jsx, etc.

    assets(folder inside src folder):
    ye folder bhi public folder ki tarah hota hai isme hum apni logo filles ya kuch or rakh sakte hain.(depands on you aap kasie use karte ho.).

- Ab hum dekhte hain files kon kon si hain or kaise use karni hai:
    hamari app banti hai app.jsx file me jo connect hoti hai main.jsx file se, and main.jsx file index.html file ko control karti hai. Index.html(grandparent)==>>main.jsx(parent)==>>app.jsx(child).

- "<> </>" in tags ko fragments kehte hain jo k hum app.jsx file mein use karte hain, ye basically ek div tag ki tarah kaam karte hain jiski zaroorat humein react likhne me lagti hai.

- eslint.config.js:
    Ye file hum use karte hain custom rules banane ke liye, for example:
    jab hum contibution ke taur par kaam karte hain to har banda alag tarah code likhta hai koi ";" use karta hai koi nhi karta, to iski wajah se sometimes compilation ke time par error phase karne pad sakte hain.
    isi problem ko solve karne ke liye hum custom rules banate hain take hamari har files properly run ho sake.(aage hum iska bhi use karne wale hain)

- .gitignore file:
    ye file hamari help karti hai humein konsi file github pe push karni hai konsi nahi karni hai is kaam me for example:
    node_modules folder me bohot saari files hoti hain jo hamare kisi kaam me bhi nhi aati to mein nhi chahta itni saari files ko mein github pe jaake upload karu to mein .gitignore file mein jake likh dunga mujhe ye files       upload nhi karni hai to wo nhi hongi.

- Package.json file:
    is file mein hamari saari configurations hoti hain like dependencies kon kon si use kar rahe hain wo saari isi file me hoti hai.

- package-lock.json file:
    ye bhi package.json file ki tarah hi hai bas isme or details me humein cheezein dekhne ko milti hain.

- vite.config.js file:
    is file me vite jo hamara bundler hai uski configurations hoti hain.(aage jaake humein or pata chalega iske bare me)

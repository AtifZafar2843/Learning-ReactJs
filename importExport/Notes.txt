To aaj mene padha hai Import and Export ke bare me.

- humein jab bhi kisi doosri file ko connect karna ho apni main file se(ya kisi bhi file se) tab hum Import and Export use karte hain.
- export 2 tarah ke hote hain, Named and Default export.
    Named export jab humein kisi naam ke sath(variable ka naam) kuch export karana ho tab use karte hain.
    Jab bhi mujhe Named wale export ko import karna hoga then uske liye mein ye syntax ka use karunga:
    import {age, enrollNum} from "./app.js";(curly braces and name ka khaas dhyaan rakhna hai isme).

    Default export jab humein koi ek cheez by Default export karani ho tab use karte hain.
    mein chahein kuch bhi naam dedu lekin jo mene default export kara hoga wahi value import hogi(aap mera js ka example dekh sakte hain.).

Mene dono export ko code karke bhi dikhaya hai you can see in js files.
